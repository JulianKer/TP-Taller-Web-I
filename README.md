### 1. Como iniciar el proyecto
```shell
$ mvn clean jetty:run
# http://localhost:8080/spring
```

### 6. Como correr las pruebas de punta a punta

#### Iniciar el servidor
```shell
$ mvn clean jetty:run
```
#### Correr las pruebas en otra terminal
```shell
$ mvn test -Dtest="VistaLoginE2E"
```
